# 📊 SQL Data Analytics Portfolio

Selamat datang di repositori portofolio SQL saya yang dibuat menggunakan mySQL. Di sini Anda dapat menemukan berbagai proyek cleansing dan analisis data yang telah saya kerjakan.

## 📁 Daftar Proyek

### 1. [World Layoff Analysis]
* **Deskripsi:** Data Cleansing dan analisis tren PHK massal global menggunakan SQL untuk melihat industri dan negara yang paling terdampak.


### 2. [Car Sales Analysis]
* **Deskripsi:** Data Cleansing data penjualan mobil tahun 2022-2023 yang berisi tipe, transmisi, daerah penjualan hingga warna kendaraan.


### 3. [Coffee Sales Analysis]
* **Deskripsi:** Data Cleansing data transaksi vending machine kopi yang berisi metode pembayaran,tipe produk terjual serta tanggal dan jam penjualan.
